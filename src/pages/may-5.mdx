I remember coming across the Van de Graaf Canon some years ago, from an article entitled [“The Secret Law of Page Harmony”][ph] on Retinart.

I’m fascinated by the prospect of rich layouts using CSS grids that we more commonly associate with print design.

Granted, using ratios and proportions for physical printed pages doesn’t make sense given the different context, but I wanted to see if it would be possible to use CSS Grid to visualise page construction canons.

Turns out it’s quite possible
